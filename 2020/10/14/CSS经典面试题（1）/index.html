CSS经典面试题（1）

CSS经典面试题

收录一下面试可能涉及的知识点。

  1. W3C标准盒子模型：content-box

    宽度 = (content) + border + padding + margin

    IE传统盒子模型：border-box

  2. CSS选择器

    id选择器（#）、类选择器（.）、标签选择器（div）、相邻选择器（+）、子选择器（>）、后代选择器（ ）、通配符选择器（*）、属性选择器（a[rel=”external”]）、伪类选择器（:hover、:nth-child）

  3. 可继承的属性：font-size，font-family，color

    不可继承的属性：border，padding，margin，width，height

  4. CSS优先级：

    !important > 行内样式> id > 类 > 标签 > 通配符 > 继承 >浏览器默认属性

  5. CSS3新增伪类：

    p:first-of-type 属于其父元素的首个元素

    p:last-of-type 属于其父元素的最后元素

    p:only-of-type 属于其父元素唯一的元素

    p:only-child 属于其父元素的唯一子元素

    p:nth-child(2) 属于其父元素的第二个子元素

    :enabled :disabled 表单控件的禁用状态

    :checked 单选框或复选框被选中

  6. CSS实现垂直居中（水平居中的方法 margin: 0 auto;）

    （1）translate变换；（2）flex布局；（3）绝对定位与top/bottom/left/right: 0; ；（4）::before伪类

    .box{
    position: relative;
	top: 50%;
	transform: translateY(-50%);
}
    .parent{
  display: flex;
  align-items: center;
}
    .box{
	margin: auto;
	position: absolute;
	top: 0; bottom: 0; left: 0; right: 0;
}
    .parent::before{
	display: inline-block;
	content: "";
	vertical-align: middle;
	height: 100%;
}

  7. display有哪些类型？

    inline - 内联； none - 隐藏； block - 块显示；table - 表格显示； list-item - 项目列表；inline-block - 内联块

  8. position的值

    static（默认）- 按照正常文档流进行排列

    relative（相对定位）- 不脱离文档流，参考自身静态位置通过top等定位

    absolute（绝对定位）- 参考距其最近的一个不为static定位的父级元素通过top等定位

    fixed（固定定位）- 所固定的参照对象是可视窗口

  9. CSS3新特性：

    RGBA和透明度（Alpha）；

    background-image，background-origin，background-size background-repeat；

    word-wrap（对不可分割的长单词换行），word-wrap: break-word;；

    文字阴影：text-shadow: 5px 5px 5px #FF0000;（水平阴影，垂直阴影，模糊距离，颜色）

    font-face：自定义字体；

    border-radius；

    边框图片：border-image: url(src) 30 30 round;

    盒阴影：box-shadow: 10px 10px 5px #FF0000;

    媒体查询：@media screen and (max-width:xxxpx){}

  10. flex布局单独写一章博客

  11. display属性规定元素应该生成的框的类型；position属性规定元素的定位类型；float属性是一种布局方式，定义元素在哪个方向浮动。

  12. BFC（块级格式化上下文）规定了内部的块级盒子如何布局，适用场景：解决边距折叠；自适应双栏布局；解决高度塌缩。

  13. CSS优化/提高性能的方法有哪些？

    避免过度约束；

    避免后代选择符；

    避免链式选择符；

    使用紧凑的语法；

    避免不必要的命名空间；

    避免不必要的重复；

    使用表示语义的名字；

    避免!important；

  14. 浏览器是怎样解析CSS选择器的？

    CSS选择器的解析是从右到左解析的，先找到所有最后节点，对于每一个节点，向上寻找其父节点，直到找到根元素或满足条件的匹配规则，则结束这个分支的遍历。

    在CSS解析完毕后，需要将解析的结果与DOM tree的内容一起进行分析，建立Render tree用来渲染。

  15. margin和padding的适用场景：

    margin：在border外侧添加空白；空白处不需要背景色；上下相连的margin可相互折叠。

    padding：在border内侧添加空白；空白处需要背景色；padding不可折叠。

  16. 响应式设计是指一个网站能够兼容多个终端，基本原理是通过媒体查询检测不同的设备屏幕尺寸进行处理。

    页面头部必须有viewport声明。

  17. CSS Sprites

    将一个页面涉及到的所有图片都包含到一张大图中，然后利用CSS的background-image，background-repeat，background-position的组合进行背景定位。利用CSS Sprites能够很好地减少网页的http请求，减少图片的字节，从而提高页面的性能。
