this 关键词

this 关键词

本文仅讨论函数的内部属性 this在 JavaScript 中的表现

this引用的是函数执行的环境对象。

call和apply

每个函数都包含两个非继承而来的方法：apply()和call()。这两个方法的用途都是在特定的作用域中调用函数，实际上等于设置函数体内this对象的值。

使用这两个方法扩充函数的作用域的最大好处，就是对象不需要与方法有任何耦合关系。

函数作用域 参数数组
apply() 可以是Array的实例，也可以是arguments对象。
call() 必须明确地传入每一个参数，逐个列举出来

bind

ES5定义了bind()方法，创建一个函数的实例，其this值会被绑定到传给bind()函数的值。例如：

window.color = "red";
var o = { color: "blue" };

function sayColor(){
    alert(this.color);
}
var objectSayColor = sayColor.bind(o);
objectSayColor();  //blue

即使是在全局作用域中调用objectSayColor()，也能得到 ”blue“ 结果，因为它的this值等于o。

简单的bind()实现：

function bind(fn, context){
    return function(){
        return fn.apply(context, arguments);
    };
}

在bind()中创建一个闭包，闭包使用apply()调用传入的函数，并传递context对象（环境）和内部函数（本例中为fn）的arguments参数。

JS的原生bind()可以直接在函数上调用这个方法，即function.bind(context)

bind()的应用场景：

将某个函数指针以值的形式传递，同时该函数必须在特定环境中执行。

被绑定函数主要用于事件处理程序以及setTimeout()和setInterval()。

apply/call 与 bind 的区别

bind()适用于回调执行，而apply()或call()会立即执行。
