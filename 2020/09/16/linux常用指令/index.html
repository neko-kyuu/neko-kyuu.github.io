linux常用指令

linux常用指令

  1. 查看当前路径 pwd

  2. 清屏： clear

  3. 退出当前命令： ctrl+c 彻底退出

  4. 执行睡眠 ： ctrl+z 挂起当前进程fg 恢复后台

  5. 查看当前用户 id： id / id 用户名

    [root@localhost ~]# id lamp
 uid=501(lamp) gid=501(lamp) groups=501(lamp)
#能看到uid(用户ID)、gid(初始组ID), groups是用户所在组，这里既可以看到初始组，如果有附加组，则也能看到附加组

  6. 查看指定帮助用什么命令？

    在任意命令后接 –help参数就可以看到该命令的简要提示

    使用man 加命令可以查看某个命令的详细帮助文档

  7. 创建目录 mkdir 创建文件 touch 复制文件 cp

  8. 文件权限修改 chmod 751 file

    7（读/写/执行）

    5（读/执行）

    1（执行）

    chmodu+xfile给file的属主增加执行权限 chmod 751 file 给 file 的属主分配读、写、执行(7)的权限，给 file 的所在组分配读、执行(5)的权限，给其他用户分配执行(1)的权限
    chmodu=rwx,g=rx,o=xfile上例的另一种形式 chmod =r file 为所有用户分配读权限
    chmod444file同上例 chmod a-wx,a+r file同上例
    $ chmod -R u+r directory 递归地给 directory 目录下所有文件和子目录的属主分配读的权限

  9. 终端是哪个文件夹下的哪个文件？黑洞文件是哪个文件夹下的哪个命令？

    终端 /dev/tty

    黑洞文件 /dev/null

  10. 删除文件 rm

    连目录及目录下文件一起删除呢 rm -r

    删除空文件夹 rmdir

  11. 用什么命令对一个文件的内容进行统计？(行号、单词数、字节数)

    wc 命令 - c 统计字节数 - l 统计行数 - w 统计字数。

  12. 查看后台任务： job -l

  13. 把后台任务调到前台执行：fg

    把停下的后台任务在后台执行起来：bg

  14. 搜索文件用什么命令? 格式是怎么样的?

    find <指定目录> <指定条件> <指定动作>

    whereis 加参数与文件名

    locate 只加文件名

    find 直接搜索磁盘，较慢。

    find / -name “string*”

  15. 使用什么命令查看用过的命令列表? history

  16. 使用什么命令查看网络是否连通? netstat

  17. 使用什么命令查看 ip 地址及接口信息？ ifconfig
